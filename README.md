# app-nodejs

Repo app nodejs

## Requisitos

- Node.js 22+

## Scripts

- `npm run dev` — roda a API em modo desenvolvimento
- `npm run build` — gera o pacote em `/dist`
- `npm start` — executa a API a partir de `/dist`

## Estrutura

- `src/` — código-fonte
- `dist/` — build final

## API

- `GET /` — retorna `{ "message": "Hello, Node.js 22 API!" }`
